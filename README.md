# Gleam JS Todo List

Two versions of the same app, one in Gleam and one in JavaScript.

## Gleam

Commands:

```bash
cd todo_list_gleam

gleam run add "first todo"
gleam run list
gleam run clear
gleam run done 1
```

## JavaScript

Commands:

```bash
cd todo_list_js

node todo.mjs add "first todo"
node todo.mjs list
node todo.mjs clear
node todo.mjs done 1
```
